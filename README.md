# Shivani Mitra Portfolio

Premium recruiter-ready portfolio for Shivani Mitra, built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Content updates

All editable portfolio content lives in `data/portfolio.ts`, including personal details, metrics, brands, expertise, case studies, career journey, tools, and selected work links.

Campaign sections intentionally use bracketed placeholders where verified objectives, creative, or metrics were not available in the resume/work notes.
